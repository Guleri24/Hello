## Requirments

* Java 17+
* Gradle (optional)
* Kotlin

## Running from Source

1. Clone the repo

2. Run

    `gradle bootRun` 

Using Gradle wrapper:

    ./gradlew bootRun (Unix)
    
    .\gradlew.cmd bootRun (Windows)

3. Search 
    
    `localhost:8080`
    `localhost:8080?name=guleri`
